# Calculator App 

A clean and responsive calculator built using HTML, CSS, and JavaScript.

## Features

- Basic operations: +, -, *, /
- Decimal support
- Clear function
- Responsive layout
- Keyboard-friendly interface (optional future upgrade)

## Live Demo

[Open Calculator](https://ivan-grozni-2.github.io/calculator-app/)

## Tech Stack

- HTML5
- CSS3 (Grid layout)
- JavaScript (ES6)
